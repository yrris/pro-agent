# 风格模板明细（L3 参考）

每个模板给出：完整风格串（直接拼进 prompt 的风格/光影/质量段）与适用场景。

## 写实摄影
- 风格串：`写实照片，85mm 定焦人像镜头，f/1.8 浅景深，自然窗光，皮肤质感真实，RAW 直出效果，超高细节`
- 场景：人像、产品图、真实感封面。

## 水墨国风
- 风格串：`中国传统水墨画，大面积留白，墨色五阶浓淡，飞白笔触，宣纸纹理，印章点缀，意境悠远`
- 场景：诗词配图、文化主题、书签海报。

## 赛博朋克
- 风格串：`赛博朋克风格，霓虹粉蓝紫灯光，雨夜反光街道，全息广告牌，高对比高饱和，电影感构图`
- 场景：科技海报、游戏氛围图。

## 皮克斯 3D
- 风格串：`皮克斯风格 3D 渲染，角色圆润可爱，大眼睛，鲜艳明快色彩，柔和全局光照，次表面散射皮肤`
- 场景：儿童向、吉祥物、头像。

## 扁平插画
- 风格串：`扁平矢量插画，几何色块构成，2-4 色有限配色，简洁粗线条，负空间构图，现代设计感`
- 场景：banner、图标化场景、PPT 配图。

## 像素风
- 风格串：`16-bit 像素艺术，复古街机游戏画面，有限调色板，抖动渐变，等距视角`
- 场景：游戏素材、怀旧主题。
